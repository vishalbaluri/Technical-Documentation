Overview

Java is an advanced, class-based, object-oriented language crafted to minimize implementation dependencies.

It powers large-scale enterprise software across platforms.

Because Java compiles to bytecode, it runs wherever a compatible JVM exists—no recompilation needed.

Basic Syntax

Drawing heavily from C++, Java’s syntax requires each program to reside in a class.

Sample “Hello, World!”:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

The entry point is always the main method inside a class.

Data Types

Java categorizes types into primitive and reference types:

  • Primitive: int, byte, short, long, float, double, char, boolean
  • Reference: String, Array, Class, Interface

Flow Control

Control structures in Java include conditionals, loops, and switch statements.

if (x > 0) {
    System.out.println("Positive");
}

Typical statements: if, else, for, while, and switch.

OOP Concepts

Java is fundamentally object-oriented. Core principles:

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

Exception Handling

Java lets you manage runtime errors using try, catch, finally, and throw statements.

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
} finally {
    System.out.println("Execution complete.");
}

Generics

Generics enable you to write type-safe code by parameterizing classes and methods.

List<String> names = new ArrayList<>();
names.add("Alice");
String first = names.get(0);

Collections Framework

The Collections Framework provides interfaces and implementations for storing groups of objects.

  • List: Ordered, allows duplicates (e.g., ArrayList).
  • Set: No duplicates (e.g., HashSet).
  • Map: Key–value pairs (e.g., HashMap).

Streams API

The Streams API (since Java 8) supports functional-style operations on collections.

List<Integer> nums = List.of(1,2,3,4,5);
nums.stream()
    .filter(n -> n % 2 == 0)
    .forEach(System.out::println);

Lambda Expressions

Lambdas provide a concise way to represent anonymous functions:

Runnable r = () -> System.out.println("Running in a lambda!");
new Thread(r).start();
